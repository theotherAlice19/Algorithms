## Recursion
No garbage collector, manual memory awareness, and a call stack you have to respect. Recursion here isn't abstract — you feel it. This repo is where I pushed through the confusion and started seeing recursive thinking as a real tool.

## What I worked on
- Base cases and recursive cases
- Factorial and Fibonacci in C
- Recursive array traversal
- Understanding the call stack in practice
- When recursion is cleaner than a loop
- Avoiding stack overflow with proper base cases
