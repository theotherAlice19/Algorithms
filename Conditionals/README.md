## Conditionals
This repo is where I got comfortable with control flow: knowing when to check something, how to combine conditions cleanly, and how to avoid the kind of logic bugs that are hard to spot but easy to make.

## What I worked on
- if, else if, else in C
- Boolean expressions with &&, ||, !
- switch statements
- Nested conditions and when to avoid them
- Comparing values correctly (including the classic = vs == mistake)
