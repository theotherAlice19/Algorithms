## Algorithms & Problem Solving
Algorithm is a set of finite, well-defined steps or instructions designed to solve a problem or perform a computation. It can also be defined as a procedure for solving a mathematical or computational problem in a finite number of steps, often involving repetitive or recursive operations.
Soo, I started learning algorithms not just to write code — but to think better.
Every problem I solve here sharpens my logical thinking and pushes my problem-solving skills further. This repo is my personal training ground.

## Topics Covered

Topics Covered
1. Arrays — the foundation of almost everything

-What is an array and why use it
-Accessing elements by index
-Modifying elements (adding, updating, removing)
-Iterating over an array
-Common array methods (push, pop, slice, etc.)

2. Conditionals — making decisions in code

-if, else if, else statements
-Comparison operators (==, !=, >, <, >=, <=)
-Logical operators (AND, OR, NOT)
-Nested conditionals
-switch / match statements
-Truthy and falsy values

3. Functions — breaking problems into clean, reusable pieces

-What is a function and why use it
-Defining and calling a function
-Parameters vs. arguments
-Return values
-Scope (local vs. global variables)
-Built-in vs. user-defined functions

4. Loops — repetition with purpose

-Why loops exist
-for loops (counted repetition)
-while loops (condition-based repetition)
-Loop control: break and continue
-Looping through arrays
-Avoiding infinite loops

5. Recursion — solving problems by solving smaller versions of themselves

-What makes a function recursive
-Base case vs. recursive case
-How the call stack works
-Simple examples (factorial, countdown)
-Recursion vs. loops — when to use which
-Common beginner mistakes

6. Strings — manipulating and understanding text-based data

-What a string is (characters + indexing)
-Common string methods (length, slice, replace, etc.)
-String concatenation and interpolation
-Searching within strings
-Looping through a string
-Converting between strings and other types
