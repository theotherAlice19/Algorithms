## Algo — C Programming Exercises

A personal collection of algorithm and logic exercises written in C, organized by topic. Each section targets a fundamental programming concept — built and solved from scratch as part of a self-driven learning path.

## What Pojects I Build?

- **01 — Arrays** — The foundation of almost everything
- **02 — Conditionals** — Making decisions in code
- **03 — Functions** — Breaking problems into clean, reusable pieces
- **04 — Loops** — Repetition with purpose
- **05 — Recursion** — Solving problems by solving smaller versions of themselves
- **06 — Strings** — Manipulating and understanding text-based data

## What Concepts I Practiced?

- **Arrays** — storing, accessing, modifying, and iterating over collections of data
- **Conditionals** — if/else logic, comparison and logical operators, switch statements
- **Functions** — defining, calling, parameters, return values, and scope
- **Loops** — for loops, while loops, break/continue, and avoiding infinite loops
- **Recursion** — base cases, recursive calls, the call stack, and factorial/fibonacci patterns
- **Strings** — character indexing, string functions, searching, and type conversion

## With What I Built With?

- C (C99)
- Self-written exercises — no external problem sets
